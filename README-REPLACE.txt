ELEGANZA RENTALS — CONTACT US + BOOK NOW UPDATE

This package contains only updated/new files.

HOW TO INSTALL
1. Stop the local PHP server with Ctrl + C.
2. Extract this ZIP.
3. Copy everything inside this extracted folder into your existing Eleganza project root.
4. Choose "Replace the files in the destination" when Windows asks.
5. Restart your PHP server.
6. Open the website and hard-refresh with Ctrl + F5.

WHAT CHANGED
- Contact Us is now a general contact/support page.
- Book Now is now a separate page at /book.php.
- The Book Now form requires a vehicle and preferred dates.
- Selecting an unavailable car shows a clear live notice.
- Selecting dates that overlap an On Hire, Reserved, Maintenance or Unavailable period shows a warning and prevents submission.
- Available dates show a positive notice, but the request still requires final confirmation.
- Car pages and homepage car cards now open Book Now with the selected car pre-filled.
- Admin Messages can distinguish Booking Requests from Contact Messages.
- CSV export and email notifications include the request type and additional booking details.

IMPORTANT
- Do not replace your storage folder. This ZIP does not contain storage files, so your cars, messages, settings and availability remain unchanged.
